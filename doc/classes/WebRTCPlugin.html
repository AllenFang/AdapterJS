pluginNeededButNotInstalledCb

() private

Defined in source/temasys.plugin.adapter.js:221

Available since 0.10.5

!!!! WARNING: DO NOT OVERRIDE THIS FUNCTION. !!! This function will be called when plugin is ready. It sends necessary details to the plugin. The function will wait for the document to be ready and the set the plugin state to AdapterJS.WebRTCPlugin.PLUGIN_STATES.READY, indicating that it can start being requested. This function is not in the IE/Safari condition brackets so that TemPluginLoaded function might be called on Chrome/Firefox. This function is the only private function that is not encapsulated to allow the plugin method to be called.

PLUGIN_LOG_LEVELS

JSON private

Defined in source/temasys.plugin.adapter.js:165

Available since 0.10.5

Log levels for the plugin. Log outputs are prefixed in some cases. From the least verbose to the most verbose

Sub-properties:

  • NONE String

    No log level.

  • ERROR String

    Errors originating from within the plugin.

  • INFO String

    Information reported by the plugin.

  • VERBOSE String

    Verbose mode.

  • SENSITIVE String

    Sensitive mode.
